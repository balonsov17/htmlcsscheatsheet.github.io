HTML and CSS Cheatsheet

HTML Tables

When creating tables in html use <table> as the wrap around tag for your whole table and its contents. In the table below are the rest of the tags used to format tables better.

HTML Table Tags/Atttributes

Tag Name Description
<table> Table The Wrapper element for all HTML tables
<head> Table Head The set of rows defining the column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row containers
<td> Table Data The table row containers
<tfoot> Table Foot The set of rows defining the footer in a table
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span

HTML Tags

Tag Name Description
<html> HTML element signifying the start and end of the document
<h1-6> Heading 1-6 heading tags ranging from 1 (the biggest) to 6 (the smallest)
<p> Paragraph signals the start of plain text paragraph
<b> Bold Will turn plain text in a paragraph bold
<i> Italic Will italcize plain text in a paragraph
<ul> Unordered list Will create an unordered (bulleted) list in html doc
<ol> Ordered list Will create an ordered (numbered)list in the html doc
<li> List Item Used within the <ol> tag to specify the ordered list items (1,2,3...)
<form> Form Used to create a user input box for forms and log-in's
<table> Table Creates a table which you can put rows columns and data into using specific html tags
<a href="url"> Link Tag Links and references to different documents (CSS) or url's
<img src="url"> Image Tag links and refences to different documents or urls for a specific picture
<src="url"> Source Path links and references to different documents or url's such as CSS
<br> Line Break will create a "break" (line of space) between paragraphs or other elements
<hr> Horizontal Line Will create a horizontal line that can be altered to meet specific needs (color, thickness, outline...)
